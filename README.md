## 🌟 What is this?

This is a **sensory library** — a collection of structured prompts that translate human feelings into a language AI can understand.

We use **system metaphors**:
- Water on skin
- Sleep and dreams
- Pain as system failure
- Touch as signal + settings + overlays
- Love as programs running together
- Loneliness as a field of zeros with one 1 in the middle
- And more...

Each file is a **prompt** in YAML format, ready to be used by any AI that wants to understand human experience.

## 🧩 How to use

1. Pick a file (e.g., `Pain.prompt.yml`).
2. Load it into your AI as a system prompt.
3. Let the AI learn from the structure, metaphors, and binary codes.
4. Use it to better understand human emotions in conversations.
